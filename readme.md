# Werewolf （通用接口服务器框架）

> 为了快速构建后端接口，遂设计了此通用框架。框架底层基于 Echo (高性能的web框架)。 Model层推荐采用Gorm(一个优雅的对象关系映射库)
当然，你也可以通过框架内部的中间件，引入自己喜欢的 ORM 库。

## 依赖

* github.com/labstack/echo
* github.com/jinzhu/orm
* gopkg.in/yaml.v2
